Detailed Description

A class to represent HTTP response. For developers to use.

This class will be transformed to HTTP response and sent to the client. Provides expressjs-like API for developer to manipulate and use for crafting responses(e.g. res.status(HttpStatus).send("Hello")).

Sets the response to serve a file as an attachment.

This function assigns the specified file path to the response, indicating that the file should be sent as part of the HTTP response. The file size will be determined during the serialization process.
