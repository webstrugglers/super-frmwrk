Detailed Description

Class used for routing.

Router uses std::unordered_map internally to map HTTP request method and path to a developer defined handler function.

Configures the router to serve static files from a specified directory.

This method sets up the router to serve static files located in the given directory and its subdirectories. It creates routes for each file, allowing them to be accessed via HTTP GET requests. Calling this method multiple times will result in undefined behaviour.

Parameters
pathThe directory containing the static files to be served.
